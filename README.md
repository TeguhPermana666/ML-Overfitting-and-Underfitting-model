# ML-Overfitting-and-Underfitting-model
Overfitting and Underfitting model di implementasikan pada model in sample scores dan train and testing split model untuk  mengetahui kualitas dari sebuah model dalam implementasi pada MAE dalam metriks sehingga di ketahui model mana yang berjalan pada under dan over.Ditunjukan bahwa kedua model dapat mecapai over tapi pada in sample scores ditunjukan turunan pada train data jadi untuk data yang baru akan menghasilkan data dari pemodelan yang tidak akurat , hal ini berbanding terbalik pada train-testing split model yang menghasilkan efektivitas yang baik dengan sedikitnya MAE dengan banyaknya model melakukan validasi pada perulangan, pada awal validasi model dihasilkan data yang yg kurng mendekati prediksi dikarenakan data yang di train dan testing train_ x dan train y di implmentasikan testing pada val_x dan val_y yang berarti kedua data tidak mengenal maka dari MAE akan semakin besar ->underfitting namun seiring model mengenal sebuah data maka MAE akan semakin kecil yang mendadakan bahwa model semakin mengenal sebuah data ->overfitting
